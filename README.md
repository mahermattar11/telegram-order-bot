# OrderlyBot - Telegram Order Management Bot

بوت تليجرام لإدارة طلبات التجار الصغار (مطاعم وملابس)

## الميزات:
- دعم لغتين (عربي/إنجليزي)
- قاعدة بيانات PostgreSQL
- لوحة تحكم ويب للتجار
- إشعارات فورية

## التثبيت:
1. انسخ `.env.example` إلى `.env`
2. املأ المتغيرات
3. pip install -r requirements.txt
4. python OrderlyBot.py
